# 🎉 Mailper 🎉

개인 맞춤형 AI 메일 비서

---

## 📌 프로젝트 개요

- **프로젝트명:** Mailper
- **목적:** 사용자가 간단한 입력만으로도 목적에 적합한 이메일을 생성 및 전송할 수 있는 자동화 서비스 개발
- **주요 타겟:** 메일 작성이 어려운 신입생, 사회초년생

### 💡 개발 동기

다른 메신저에 비해 공적인 느낌이 드는 메일 작성에 어려움을 겪는 모든 초년생들에게 간단한 요구사항 입력만으로 목적에 맞는 완벽한 메일을 생성 및 전달까지 가능케 합니다.

---

## 🌟 주요 기능

### 1. 💬  AI 메일 생성 기능

- **설명:** 메일을 보내는 상황, 원하는 메일 분위기(정중, 친근, 간결 등), 메일 송신자 이름/메일주소 등 간단한 입력 후 '생성하기' 버튼 클릭시 AI가 자동으로 프롬프트를 학습하여 메일 생성
- **기술:** OpenAI ChatGPT API

  
### 2. 💬  메일 전송 기능

- **설명:** 가입한 구글 계정을 수신자로 설정해 서비스 내에서 메일을 대신 전송
- **기술:** Java MailSender, Google OAuth2

---

## 🔧 기술 스택

- **백엔드:** Spring Boot, Spring Data JPA, JDK 17, OpenAI Assistant API
- **프론트엔드:** Javascript, React
